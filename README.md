# Simple Flask App

This project is a basic Flask web application.

## Features

- Home page route (`/`)
- Simple response

## Requirements

- Python 3.x
- Flask

## Installation

```bash
pip install flask
```

## Usage

```bash
python app.py
```

Visit [http://localhost:5000](http://localhost:5000) in your browser.
